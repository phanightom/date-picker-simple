## Search Repos Project

### Getting Started

First, run the development server:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Feature
1. 專案模組化
2. 搜尋 Repos 項目
3. 無限滾動顯示項目資料
4. 搜尋中狀態顯示 & 搜尋結果顯示

### Skills
1. ReactJs
2. NextJs
3. TypeScript
4. Styled-Component
5. Lodash
6. React-Loading
